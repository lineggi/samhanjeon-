# 세력별 의상·갑옷 레퍼런스 (삼한전 초상 생성 기준)

각 세력 영웅 초상은 **이 문서의 고증 + `assets/refs/<세력>/`에 넣은 레퍼런스 이미지**를 기반으로 생성한다.
`tools/gen-portraits.mjs`가 해당 폴더에 이미지가 있으면 **그 이미지를 참조(edits)** 해서 생성하고,
없으면 아래 텍스트 묘사(FAC_STYLE)만으로 생성한다.

## 사용 방법
1. 아래 "추천 레퍼런스"를 검색해 실제 유물·고분벽화·박물관 복원 사진을 모은다.
2. 세력별로 **2~4장**을 `assets/refs/<세력>/`에 넣는다. (예: `assets/refs/goguryeo/gaemamusa1.jpg`)
   - 폴더: `goguryeo · baekje · silla · gaya · china · wa`
   - 권장: 정면/측면 인물, 갑옷 전체가 보이는 컷, 투구 클로즈업
3. 생성: `node tools/gen-portraits.mjs --only 온달,소수림왕` → 로그에 `(ref:N)` 표시되면 레퍼런스 기반 생성.
   - 레퍼런스가 없으면 텍스트 묘사로 생성된다.
4. 의상 기준을 바꾸려면 이 문서를 고치고, 동일 내용을 `tools/gen-portraits.mjs`의 `FAC_STYLE`(중국은 `chinaStyle`)에 반영한다.

> 저작권: 박물관 소장 유물·고분벽화 등 **공개·저작권 만료** 자료를 레퍼런스로 사용한다. 특정 작가의 일러스트/게임 원화·드라마 스틸은 피한다.

## 한국 갑옷 유형 (고증 기본)
삼국시대 갑옷은 크게 두 종류다. (출처: 나무위키 "삼국시대 갑옷", 한국민족문화대백과 "고대갑주/판갑", 우리역사넷)
- **찰갑(札甲)**: 작은 갑찰(소찰)을 가로로 잇고 다시 세로로 끈으로 엮은 **유동성 있는 비늘형 갑옷**. 기마전투에 적합. → **고구려·백제** 주력.
- **판갑(板甲)**: 여러 모양의 **큰 철판**을 둥근 못(리벳)이나 가죽끈으로 결합한 **단단한 판형 갑옷**. 보병전투에 유리. → **신라·가야**에 많음(특히 **종장판갑** = 세로로 긴 철판, 4~5세기 영남 지역).
- **투구 종류**: 종장판주(세로판) · 만곡종장판주(곡선 세로판, 고구려) · 소찰주(작은 찰) · 차양주(챙 달린 모자형) · 충각부주(앞이 뿔처럼 돌출).
- **마갑(馬甲)/개마무사**: 말에 입히는 갑옷 + 중장기병. 고구려가 대표(안악3호분 행렬도), 가야도 마갑총 출토.
- **시대 변화**: 신라·가야는 4~5세기 초 **판갑** 위주였다가 **광개토대왕 남정(400) 이후 고구려식 찰갑을 도입**해 갔다. 따라서 645·554 시대의 신라는 찰갑 비중이 높고, 400·371 시대의 가야·신라는 판갑이 자연스럽다.

## 고구려 (Goguryeo)
- **핵심 의상**: 작은 세로 철편을 **붉은 끈**으로 엮은 은빛 찰갑(札甲), 어깨·치마 찰갑, 가슴의 **사자머리 금장식**과 금 사자 허리띠, **뾰족한 철투구**(날개·사자 장식 + 붉은–검정 깃털, 목·뺨 가리개) 또는 **상투**, 진홍색 망토(금 사자 걸쇠). 색조 = 은회색 + 진홍 + 약간의 금. **파란색 금지.**
- **병종 특색**: 개마무사(鎧馬武士) — 기마+말갑옷(마갑).
- **추천 레퍼런스**: 안악3호분 행렬도(개마무사), 무용총·각저총·삼실총 벽화, 국립중앙박물관 고구려 찰갑 복원, 개마무사 복원 모형, 고구려 철제 투구·찰갑 유물.
- **레퍼런스 폴더**: `assets/refs/goguryeo/`
- **현재 프롬프트(FAC_STYLE.goguryeo)**: 작은 세로 철편 붉은 끈 찰갑 + 사자머리 금장식 + 붉은 깃털 철투구/상투 + 진홍 망토, 은회색·진홍, 파란색 배제.
- **인물별 역사 장면(SCENE)**: 온달=관을 향해 외침(설화), 소수림왕=성벽 전투, 고국원왕=평양성서 화살 피격, 고연수/고혜진=안시성 앞 진군, 고무=요동 성벽, 모두루=부여 변경.

## 백제 (Baekje)
- **핵심 의상**: 고구려와 유사한 **철 찰갑(札甲)**(무사) — 몽촌토성 출토 골제 찰갑도 있음 — 위에 **비취색·금색 비단 포(袍)**, **금동관식**(불꽃·인동 무늬, 백제금동대향로풍 투조), 우아한 궁정 복식, 붉은 망토 대신 녹색·금색 비단 띠.
- **추천 레퍼런스**: 무령왕릉 금제 관식·금동관, 백제금동대향로(국보 287호), 백제 금동신발, 능산리·무령왕릉 출토 유물, 부여 백제문화단지 복원.
- **레퍼런스 폴더**: `assets/refs/baekje/`
- **현재 프롬프트(FAC_STYLE.baekje)**: gilt-bronze 비늘갑 + 비취-금 비단포 + 불꽃 투조 금동관식 + 녹/금 비단 띠, 비취-금 배경.

## 신라 (Silla)
- **핵심 의상**: 무사는 **종장판갑(縱長板甲)** 또는 철 찰갑에 금장식 마감, 왕족은 **곡옥(曲玉)** 달린 **금관/금동관**(출(出)자형 입식)·금 허리띠(과대)·금 귀걸이, 진홍–금색 망토. (신라는 판갑·찰갑 병용)
- **추천 레퍼런스**: 천마총·황남대총·금관총 금관, 금제 과대, 곡옥, 기마인물형토기, 국립경주박물관 금공예.
- **레퍼런스 폴더**: `assets/refs/silla/`
- **현재 프롬프트(FAC_STYLE.silla)**: 도금 갑옷 + 금장식·곡옥 + 출자형 금관/금동관 + 진홍-금 망토, 황금-호박 배경.

## 가야 (Gaya)
- **핵심 의상**: 실용적 **종장판갑(縱長板甲)** — 세로로 긴 큰 철판을 리벳/가죽끈으로 결합 — + 철투구(**종장판주·차양주·충각부주**류), 가죽끈, 절제된 보라/자연색 천, 망토 거의 없음. 기병은 **마갑(마갑총 출토)**. 철의 왕국 분위기.
- **추천 레퍼런스**: 대성동·복천동 고분 판갑·철투구, 마갑총 말갑옷, 김해 대성동고분박물관·국립김해박물관 철제 갑주.
- **레퍼런스 폴더**: `assets/refs/gaya/`
- **현재 프롬프트(FAC_STYLE.gaya)**: 리벳 철판갑 + 철투구 + 가죽끈 + 절제된 보라/자연색, 슬레이트-스틸 배경.

## 중국 (전연·후연·북연·북제·당) — 인물(시대)별 구분
중국은 시대마다 의상이 크게 다르다(`chinaStyle`로 인물명에 따라 분기).
- **선비 전연·후연 (모용씨·풍발)**: 모피 두른 찰갑, **땋은 머리** + 모피 모자/뾰족 투구, 가죽·모피 유목 복식. → `assets/refs/china/`에 선비 기병 자료.
  - 추천 레퍼런스: 모용선비 기병 복원, 안양 북조 벽화, 십육국~북조 기병 갑주.
- **북제 (고양·곡율광)**: 둥근 어깨 보호대의 북조식 중장 찰갑, 높은 능형 투구, 차분한 어두운 포.
  - 추천 레퍼런스: 누예묘(婁叡墓) 벽화, 북제 무사용, 북조 명광개 초기형.
- **당 (장량·이도종·이세민·이세적·설인귀)**: **명광개(明光鎧)** — 가슴의 둥근 호심경 2개, 짐승머리 견갑, 넓은 소매 관복, 날개 장식 당식 투구. 흑·주(朱)·금의 황실색.
  - 추천 레퍼런스: 소릉육준, 당삼채 무사용·천왕상, 당대 명광개 복원, 둔황 벽화 무장.
- **레퍼런스 폴더**: `assets/refs/china/` (시대 구분이 필요하면 파일명에 `tang_`, `xianbei_`, `beiqi_` 접두사로 표시 권장)

## 왜 / 야마토 (Wa / Yamato)
- **핵심 의상**: 고분시대 갑옷 — 리벳 **단갑(短甲)/괘갑(挂甲, 케이코)**, **챙/충각 달린 투구**(미비부주·충각부주) + 목가리개, 흰–붉은 끈 엮음, 흰 망토.
- **추천 레퍼런스**: 고분시대 단갑·괘갑 유물, 충각부주·미비부주, 무인 하니와(埴輪), 도쿄·나라국립박물관 고분 갑주.
- **레퍼런스 폴더**: `assets/refs/wa/`
- **현재 프롬프트(FAC_STYLE.wa)**: 리벳 케이코 흉갑 + 챙 투구 + 흰-붉은 끈 + 흰 망토, 옅은 스톤-화이트 배경.
